## Método a Corregir
Reemplazar las llamadas a métodos no estándar:
- `included()` -> `with()`
- `filter()` -> Eliminar o implementar lógica de filtrado
- `sort()` -> Eliminar o implementar lógica de ordenamiento
- `getOrPaginate()` -> `get()` o `paginate()`
