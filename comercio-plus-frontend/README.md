# ComercioPlus Frontend (React)

Documentacion oficial del proyecto:

- `../docs/UNIVERSAL_COMERCIOPLUS.md`
- `../docs/DOC_GOVERNANCE.md`

Uso rapido:

- `npm run dev` (desde raiz del repo)
- `npm run build` (desde raiz del repo)
- `npm run test:e2e` (desde raiz del repo)

Configuracion API:

- Definir `VITE_API_BASE_URL` en produccion.
- Si falta en produccion, el cliente bloquea requests de auth para evitar comportamientos inconsistentes.
